# Python Data Processing Foundations

## Overview
This project demonstrates foundational Python and NumPy concepts used in data processing workflows, including array operations, control flow, data structures, copying behavior, and exception handling.

## Skills Demonstrated
- NumPy array manipulation
- Type casting
- Conditional logic and loops
- Tuple unpacking
- List operations
- Dictionary construction
- Error handling with try/except

## Technologies
- Python 3
- NumPy

## How to Run
```bash
pip install -r requirements.txt
python src/fundamentals.py
```
